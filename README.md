# WebAPI 2.0 Practice

* Build APIs based on Resources
* Use and understand Status Codes
* Design APIs with association URIs
* Version your APIs

# Installation
* Change ConnectionString to SQL Database in Web.config
* Update SQL Database : 
```
update-database
```
